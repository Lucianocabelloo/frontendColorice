## Instalación ⚙️

<p>Para correr el proyecto primero se clonara </p>

`$ git clone "url"`

Luego nos colocaremos en la carpeta del archivo

`$ cd "carpeta"`

Una vez posicionados haremos para instalar todas las dependencias

`$ npm install`

y finalmente 

`$ npm run dev`
